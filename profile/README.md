# Nuxt Aid

Nuxt Aid is a community for developers and non-profits to learn, share, and build impactful solutions with Nuxt.

## Focus

- Nuxt modules and integrations
- Developer tooling for the Nuxt ecosystem
- Open source building blocks for real-world applications

## Featured projects

- [nuxt-mongoose](https://github.com/nuxtaid/nuxt-mongoose) - A Nuxt module for simplifying the use of Mongoose in your project.
- [nuxt-paypal](https://github.com/nuxtaid/nuxt-paypal) - A Nuxt module for simplifying the use of PayPal in your project.
- [nuxt-neo4j](https://github.com/nuxtaid/nuxt-neo4j) - A Nuxt module for integrating Neo4j into Nuxt applications.
- [nuxt-recaptcha](https://github.com/nuxtaid/nuxt-recaptcha) - Lightweight Google reCAPTCHA integration for Nuxt.
- [nugit](https://github.com/nuxtaid/nugit) - Git operations from Nuxt DevTools.

This `.github` repository provides the default community health files for the Nuxt Aid organization, including issue templates, a pull request template, and shared contribution guidance.
