# Nuxt Aid

Nuxt Aid is a community for developers and non-profits to learn, share, and build impactful solutions with Nuxt.

## Focus

- Nuxt modules and integrations
- Developer tooling for the Nuxt ecosystem
- Open source building blocks for real-world applications

## Get involved

- Browse repositories: https://github.com/orgs/nuxtaid/repositories
- Visit the website: http://nuxtaid.org
- Open issues and pull requests in the relevant repository
