# Support

## Where to ask questions

For usage questions, setup help, and implementation details, use the repository that matches the package or module you are working with.

- Start with the repository README and docs
- Open an issue in the relevant repository if the project uses issues for support
- Visit http://nuxtaid.org for organization information

## Bug reports

Use the bug report issue template and include a minimal reproduction when possible. Report bugs in the repository where the problem occurs.

## Security issues

Do not use public support channels for security reports. Follow [SECURITY.md](./SECURITY.md).
